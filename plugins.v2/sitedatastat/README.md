# 站点数据统计·自主版

插件 ID：`sitedatastat`

完全自主抓取并解析各站点用户数据，**不依赖** MoviePilot 核心的 `refresh_userdatas` 流程与 `SiteUserData` 存储。用于替换社区版「站点数据统计」(SiteStatistic)。

## 特点

- **自主采集**：插件内置 fork 了一套站点解析器（`parser/`），自己驱动抓取与解析，独立于核心解析流程。
- **修复内置**：
  - 做种通用兜底：表格列检测失配时，从做种列表页 “N条记录 … X TB” 汇总行补全做种数与做种体积。
  - 用户等级兜底：覆盖 `等级` 行（图标 title / 文本）以及仅有等级图标 title 的站点（如 hhanclub）。
  - 抗抖动：本次抓取失败或做种/等级偶发归零时，沿用上一次的好值，避免数据跳变。
  - 失败重试一次。
- **自有存储**：按日期存于插件自有数据，最多保留 90 天。
- **展示**：总览卡片（总上传/下载/做种数/做种体积）+ 各站点明细表（站点/用户名/等级/上传/下载/分享率/魔力/做种数/做种体积）。

## 配置

| 配置项 | 默认 | 说明 |
| --- | --- | --- |
| 启用插件 | 关 | |
| 立即运行一次 | 关 | 保存后立刻采集一次 |
| 失败重试一次 | 开 | 单站抓取失败时重试一次 |
| 采集周期(cron) | `30 8 * * *` | 定时采集 |
| 采集站点 | 空=全部 | 选择要采集的站点 |
| 采集后通知 | 不发送 | 今日增量 / 累计全量 |

## 说明

- 数据完全由本插件采集，与核心「站点数据」/社区版 SiteStatistic 相互独立。
- 站点的 Cookie、UA、代理、超时复用 MoviePilot 站点配置。
