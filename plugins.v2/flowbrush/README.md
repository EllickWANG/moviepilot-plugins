# 刷流

`FlowBrush` 是基于线上 `BrushFlow` 4.3.5 复刻的私人维护版，插件展示名缩短为“刷流”，配置前缀、服务 ID 与原插件隔离，避免与原 `BrushFlow` 同时存在时互相覆盖。

## 来源

- 原插件：`BrushFlow` / 站点刷流
- 原作者：jxxghp, InfinityPacer, Seed680
- 私人维护：Ellick
- 原项目：<https://github.com/InfinityPacer/MoviePilot-Plugins>
